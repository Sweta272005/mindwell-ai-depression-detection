# MindWell AI – Depression Risk Prediction System

## Overview
MindWell is an AI-based mental health support system that predicts depression risk using PHQ-9 responses, machine learning, sentiment analysis, and trend monitoring.

## Features
- PHQ-9 depression risk assessment
- Machine learning risk prediction
- Sentiment analysis using NLP
- Symptom importance analysis
- Depression trend monitoring

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- TextBlob
- Google Colab

## Workflow
PHQ Test → ML Prediction → Sentiment Analysis → Risk Adjustment → Trend Monitoring
